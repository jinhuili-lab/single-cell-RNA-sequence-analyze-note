Answers

  1. What is min.features? Describe it and min.features = 100.

Ans: The min.features argument specifies the minimum number of genes that need to be detected per cell. This argument will filter out poor quality cells that likely just have random barcodes encapsulated without any cell present. The min.features argument is usually set to 100 to detect and filter cells with less than 100 genes so that they are not considered for analysis.
